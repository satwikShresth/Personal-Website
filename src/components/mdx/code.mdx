Here's a simple utility function I use frequently in my projects:

```javascript
function debounce(func, wait = 300) {
  let timeout;
  return function(...args) {
    clearTimeout(timeout);
    timeout = setTimeout(() => func.apply(this, args), wait);
  };
}
```

This debounce function helps limit how often a function can be called, 
which is perfect for handling events like window resizing or search inputs.
Try it in your next project!
